🔍 إعدادات SEO و Search

📈 هذه الإعدادات تساعد محركات البحث (Google, Bing) على فهم موقعك وعرضه بشكل أفضل في النتائج.

🌐 Open Graph (للفيسبوك وواتساب)

📊 Google Tools (اختياري)

📌 خطوات لتظهر شعارك في نتائج Google:
1. ارفع شعارك على رابط دائم (Firebase Storage)
2. اضبط الشعار في "الشعار والهوية"
3. سجل في Google Search Console
4. أضف verification code أعلاه
5. قدم sitemap.xml (سيتم إنشاؤه تلقائياً عند الحفظ)
